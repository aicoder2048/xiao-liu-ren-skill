# 示例1: 数字输入占卜

## 用户请求
> 帮我用小六壬占卜，数字是 1,2,3，问今日运势如何？

## 运行脚本
```bash
uv run scripts/xiaoliu.py --numbers 1,2,3 --question "今日运势如何？" --format json
```

## 占卜结果

**输入数字**: [1, 2, 3]

| | 初传（前期） | 关系 | 中传（中期） | 关系 | 末传（后期） |
|---|:---:|:---:|:---:|:---:|:---:|
| **符号** | 【大安】 | | 【留连】 | | 【速喜】 |
| **五行** | 木 | 同 → | 木 | 生 → | 火 |

### 符号解读
- **大安（木）**: 长期、缓慢、稳定。求安稳大安最吉
- **留连（木）**: 停止、反复、复杂。想挽留留连是吉
- **速喜（火）**: 惊喜、快速、突然。意想不到的好事

### 五行关系
- 大安(木) → 留连(木): 同属木，相互扶持
- 留连(木) → 速喜(火): 木生火，推动发展

### AI解读要点
整体来看，此卦前期稳定（大安），中期虽有反复（留连），但最终迎来惊喜（速喜）。木生火的五行关系表示事态会自然向好的方向发展。今日运势整体偏吉，建议保持耐心，好事将至。
