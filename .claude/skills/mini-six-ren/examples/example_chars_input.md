# 示例3: 汉字笔画输入占卜

## 用户请求
> 帮我用"天地人"三个字占卜，问最近的感情运势

## 运行脚本
```bash
uv run scripts/xiaoliu.py --chars "天地人" --question "最近感情运势如何？" --format json
```

## 算法说明
汉字输入的转换规则：
1. 取前3个汉字
2. 查询每个汉字的笔画数
3. 将笔画数作为三个输入数字

例如 "天地人":
- 天 = 4画, 地 = 6画, 人 = 2画
- 数字: [4, 6, 2]

## 占卜结果

**输入**: 天(4) 地(6) 人(2)

| | 初传（前期） | 关系 | 中传（中期） | 关系 | 末传（后期） |
|---|:---:|:---:|:---:|:---:|:---:|
| **符号** | 【赤口】 | | 【天德】 | | 【留连】 |
| **五行** | 金 | 同 → | 金 | 克 → | 木 |

### AI解读要点
初传赤口在感情问题上提醒注意口舌之争，中传天德暗示可寻求长辈或贵人指点，末传留连表示感情关系会经历反复纠结。金克木的五行关系暗示过于强势的态度可能损害关系发展。建议在感情中多一些包容和耐心，减少争执。
