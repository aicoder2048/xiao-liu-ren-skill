# 小六壬九宫符号详解

## 九宫排列（左手掌诀）

```
食指:  [留连②]
        ↓
中指:  [大安①] → [速喜③]
        ↑
无名指: [天德⑨] ← [赤口④]
        ↑
中指下: [小吉⑤]
中间:   [空亡⑥]
西南:   [病符⑦]
东北:   [桃花⑧]
```

顺序: 大安→留连→速喜→赤口→小吉→空亡→病符→桃花→天德（循环）

## 五行生克关系

```
     木 → 火 → 土 → 金 → 水 → 木  （相生）
     木 → 土 → 水 → 火 → 金 → 木  （相克）
```

| 关系 | 说明 | 对事态影响 |
|:---:|---|---|
| 生 | 前者促进后者 | 顺利推进，有助发展 |
| 克 | 前者制约后者 | 存在阻碍，需克服困难 |
| 被生 | 后者反哺前者 | 有外力支持 |
| 被克 | 后者反制前者 | 受到压制 |
| 同 | 同属一行 | 相互扶持，力量加强 |
| 无 | 无直接关系 | 各自独立发展 |

## 三传解读框架

- **初传（前期）**: 事情的起始阶段，当前状态
- **中传（中期）**: 事态的发展过程，变化趋势
- **末传（后期）**: 最终结果，事态归宿

### 解读优先级
1. 先看末传（最终结果最重要）
2. 再看三传之间的五行生克关系
3. 最后结合各符号本身的含义和求问事项

### 吉凶简判
- **大吉**: 大安、速喜、天德
- **小吉**: 小吉、留连（视情况）
- **凶**: 赤口（最凶）、空亡（金钱最忌）、病符
- **中性**: 桃花（恋爱吉，其余不吉）
