# 示例2: 日期时间输入占卜

## 用户请求
> 用现在的时间帮我占一卦，问这次面试能成功吗？

## 运行脚本
```bash
uv run scripts/xiaoliu.py --datetime "2025-07-15 10:30" --question "面试能成功吗？" --format json
```

## 算法说明
日期时间输入的转换规则：
1. 将公历日期转换为农历（使用lunardate库）
2. 取 **农历月份** 作为第一个数字
3. 取 **农历日期** 作为第二个数字
4. 将小时转换为 **时辰序号**（子时=1, 丑时=2, ... 亥时=12）作为第三个数字

例如 2025-07-15 10:30:
- 农历: 六月二十一
- 数字: [6, 21, 6]（六月, 二十一日, 巳时）

## 占卜结果

| | 初传（前期） | 关系 | 中传（中期） | 关系 | 末传（后期） |
|---|:---:|:---:|:---:|:---:|:---:|
| **符号** | 【空亡】 | | 【桃花】 | | 【天德】 |
| **五行** | 土 | 同 → | 土 | 生 → | 金 |

### AI解读要点
初传空亡预示初期可能感觉没把握，中传桃花暗示面试中人际交流是关键因素，末传天德代表有贵人相助。土生金的五行关系表示付出会有实质回报。建议面试时注重表达和人际互动，可能有面试官对你格外赏识。
